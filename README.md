# AI SOC Assistant

This project is an AI-based Security Operation Center (SOC) assistant
that analyzes firewall logs, detects cyber-attacks, maps them to the
MITRE ATT&CK framework, and generates AI-based explanations and reports.

## Features
- Firewall log analysis
- Brute-force and data exfiltration detection
- MITRE ATT&CK mapping
- AI-generated explanations using LLM
- Automatic PDF report generation

## Technologies Used
- Python
- Pandas
- OpenAI GPT (LLM)
- LangChain
- FPDF, PyPDF2

## Usage
1. Upload firewall logs in CSV format
2. Run the Python script
3. View the generated SOC report
